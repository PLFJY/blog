## 主表

| 类型       | 时间意义       | 口诀                    | 结构                                              | 例句                                   |
| ---------- | -------------- | ----------------------- | ------------------------------------------------- | -------------------------------------- |
| 真实条件句 | 表事实或可能   | 主将从现                | If + 一般现在，主句 will + 原形                   | If it rains, we will stay home.        |
| 现在虚拟   | 与现在事实相反 | 主过将从一过            | If + 过去式，主句 would + 原形                    | If I were you, I would leave.          |
| 过去虚拟   | 与过去事实相反 | 主过将完从过完          | If + had + 过去分词，主句 would have + 过去分词   | If I had known, I would have told you. |
| 将来虚拟   | 表极不可能将来 | 主将从 should / were to | If + should/were to + 动词原形，主句 would + 原形 | If he should come, I would tell him.   |
| wish 句型  | 现在愿望       | wish 从过               | wish + 过去式                                     | I wish I were taller.                  |
| wish 句型  | 过去后悔       | wish 从过完             | wish + had + 过去分词                             | I wish I had gone there.               |

## If 速查：

| 主句         | 时间意义       | 从句             |
| ------------ | -------------- | ---------------- |
| 将来         | 事实或可能     | 一般现在         |
| 过去将来     | 与现在事实相反 | 一般过去         |
| 过去将来完成 | 与过去事实相反 | 过去完成         |
| 将来         | 不可能的将来   | should / were to |

## Wish 速查:

| 时间意义 | 从句     |
| -------- | -------- |
| 现在愿望 | 一般过去 |
| 过去后悔 | 过去完成 |
